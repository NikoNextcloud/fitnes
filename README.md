# IronForm — Фитнес Наръчник

Уеб приложение с упражнения по мускулни групи и видео демонстрации.

## Какво ново: зареждане от JSON по пол + мускулна група

Упражненията вече се **зареждат динамично от `lyfta_exercises_cache.json`** (7583 упражнения).
В хедъра има превключвател **♂ Мъж / ♀ Жена**, а табовете избират мускулната група.
При смяна на пол или група списъкът се презарежда автоматично.

### Как работи
- `lyfta-loader.js` чете JSON-а и групира упражненията:
  - по мускулна група чрез `body_part_id`
  - по пол чрез маркера `(male)` / `(female)` в името; неутралните упражнения се показват и за двата пола
- Показват се само упражнения с видео; до 30 на група (подредени по приоритет).
- `app.js` рендира картите и модала от тази база (`window.LYFTA_DB`).

### Връзка група → body_part_id
| Група | body_part_id |
|-------|--------------|
| Бицепс | 17, 7 |
| Трицепс | 18, 5 |
| Рамене | 6 |
| Гръб | 4 |
| Крака | 19, 20, 1, 3, 8 |
| Гърди | 2 |

## Стартиране
Приложението ползва `fetch`, затова трябва да се отвори през **локален сървър**
(не директно с двоен клик / `file://`):

```
START_IRONFORM_LYFTA.bat
```
после отвори `http://127.0.0.1:8767/index.html`.

Или с Python:
```
python -m http.server 8767
```

## Файлове
- `index.html` — структура + превключвател за пол
- `lyfta-loader.js` — зарежда и групира упражненията от JSON
- `app.js` — рендер на карти и модал
- `data.js` — резервен/локален набор (ползва се и от AI треньора)
- `coach.js` — AI треньор
- `lyfta_exercises_cache.json` — базата с упражнения
